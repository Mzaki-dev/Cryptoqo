# Cryptoqo

Cryptoqo is a responsive dashboard for tracking live cryptocurrency statistics and real-time market data. Built with Next.js, React, Tailwind CSS, and powered by the CoinGecko API, it provides traders and enthusiasts with clean charts, tables, and search capabilities.

🚀 **Live demo**: https://cryptoqo.vercel.app/

---

## Features

- Real‑time price, market cap, and percentage change charts for top coins
- Searchable coin list with pagination
- Dark/light mode support
- Mobile‑friendly layout including full‑width search on small screens
- Simple, reusable UI components built with Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18+ (or compatible)
- pnpm (preferred) or npm

### Installation

```bash
# clone the repo
git clone https://github.com/yourusername/cryptoqo.git
cd cryptoqo

# install dependencies
pnpm install
# or npm install
```

### Development

```bash
pnpm run dev
# or npm run dev
```

Then open `http://localhost:3000` in your browser.

### Build & Deployment

```bash
pnpm run build
# or npm run build
```

The app is production‑ready and can be deployed to Vercel, Netlify, etc. It’s already hosted at the URL above.

## Technologies

- Next.js 14 (app directory)
- React & TypeScript
- Tailwind CSS
- Vite for bundling
- CoinGecko API for cryptocurrency data
- React Router (used in some internal pages)
- Framer Motion for animations
